--- layout: page title: Elo Design System description: Uma linguagem de design que comunica Benner --- Temos um desafio, comunicar a mesma mensagem através de diversas soluções, com várias tecnologias, de forma clara e consistente. Essa mensagem deverá ser ponte de conexão entre todas nossas soluções a partir de uma mesma linguagem. É isso que o Elo Design System nos permitirá atingir, para que todos possamos falar a mesma linguagem através de nossas tecnologias.
Introdução
## O que é Design System São referências padronizadas de estilos e componentes, que buscam permitir uma consistência de design e experiência entre diferentes soluções. Para que desenvolvedores possam focar mais nas tecnologias aplicadas. ## Quais os objetivos dessa linguagem - consistência de design - facilitar e agilizar desenvolvimento - permitir comunicação coesa ## Design Atômico Esse modelo de representação de código facilita o desenvolvimento e referências.
