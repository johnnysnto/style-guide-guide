--- layout: page title: Elo Design System description: Uma linguagem de design que comunica Benner --- Temos um desafio, comunicar a mesma mensagem através de diversas soluções, com várias tecnologias, de forma clara e consistente. Para isso, criamos o Elo Design System, que facilitará o trabalho dos desenvolvedores com o padrão de comunicação nas soluções desenvolvidas.
Introdução
## O que é Design System Uma forma de comunicar nossos princípios, com nossa linguagem de design padrão e melhores práticas. Uma série de referências de estilos, componentes e padrões de aplicação, para que desenvolvedores possam focar mais nas **tecnologias** aplicadas em nossas soluções. Nosso design system irá funcionar como um portal de referências para os padrões de design e melhores práticas, garantindo a comunicação da identidade Benner através das interfaces das diversas soluções disponíveis para os usuários.
## Porque desenvolvemos o Design System - Agilizar o processo de desenvolvimento através de referência concreta - Padronização de interfaces nas soluções Benner - Consistência de interações dentre as soluções - Diminui a probabilidade de erros ou problemas de usabilidade - Maior integração entre as equipes de desenvolvimento, design e envolvidos no projeto - Escalabilidade de interfaces desenvolvidas - Constante atualização e evolução dos conteúdos disponíveis ## Comece a usar agora Veja como você pode iniciar a aplicar o Design System com seu time
