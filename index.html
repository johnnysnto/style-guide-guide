--- layout: page title: Elo Design System description: Uma linguagem de design que comunica Benner --- Temos um desafio, comunicar a mesma mensagem através de diversas soluções, com várias tecnologias, de forma clara e consistente. Para isso, criamos o Elo Design System, que facilitará o trabalho dos desenvolvedores com o padrão de comunicação nas soluções desenvolvidas. ## O que é Design System Uma forma de comunicar nossos princípios, com nossa linguagem de design padrão e melhores práticas. Nosso design system irá funcionar como um portal de referências para os padrões de design e melhores práticas, garantindo a comunicação da identidade Benner através das interfaces das diversas soluções disponíveis para os usuários. - Agilizar o processo de desenvolvimento através de referência concreta - Padronizar interfaces nas soluções Benner - Maior consistência de interações dentre as soluções - Diminuir a probabilidade de erros ou problemas de usabilidade - Maior integração entre as equipes de desenvolvimento, design e envolvidos no projeto - Permitir a escalabilidade de interfaces desenvolvidas - Garantir constante atualização e evolução dos conteúdos disponíveis
